# 简易密码管理器

这是一个 Chrome 扩展程序，可以安全地保存和自动填充您在网站上输入的登录凭据。

## 功能

- **自动保存**: 当您在网站上提交登录表单时，扩展程序会自动保存您的用户名和密码。
- **密码列表**: 在扩展的弹出窗口中，您可以查看所有已保存的凭据。
- **一键填充**: 当您访问已保存凭据的网站时，可以点击“填充”按钮，自动将用户名和密码填入表单。
- **凭据管理**: 您可以随时从弹出窗口中删除不再需要的凭据。

## 如何安装和测试

由于此扩展程序是本地开发的，您需要通过以下步骤在 Chrome 浏览器中手动加载它：

1.  **打开扩展程序页面**:
    *   在 Chrome 浏览器地址栏输入 `chrome://extensions` 并按回车。
    *   或者，点击浏览器右上角的三个点菜单，选择 **扩展程序** -> **管理扩展程序**。

2.  **启用开发者模式**:
    *   在扩展程序页面的右上角，找到并打开 **开发者模式** 的开关。

3.  **加载扩展程序**:
    *   启用开发者模式后，页面左上方会出现三个按钮。点击 **加载已解压的扩展程序**。
    *   在弹出的文件选择窗口中，找到并选择包含本项目的文件夹 (`chrome-pass` 文件夹)。
    *   点击“选择文件夹”按钮。

4.  **完成加载**:
    *   如果一切顺利，您应该能在扩展程序列表中看到一个新的“简易密码管理器”卡片。
    *   在浏览器的工具栏上（通常在地址栏旁边）会看到扩展程序的图标。如果没看到，可以点击拼图形状的“扩展程序”按钮，然后将“简易密码管理器”固定到工具栏。

## 如何使用

1.  **保存密码**:
    *   访问任意一个有登录功能的网站（例如 `github.com/login`）。
    *   输入您的用户名和密码，然后点击登录。
    *   扩展程序会自动将该网站的域名、用户名和密码保存下来。

2.  **查看和填充密码**:
    *   登出该网站，或者刷新登录页面。
    *   点击浏览器工具栏上的“简易密码管理器”图标，会弹出一个窗口。
    *   您应该能看到刚刚保存的网站信息。
    *   点击旁边的 **填充** 按钮。
    *   扩展程序会自动将您之前保存的用户名和密码填入到登录框中。

3.  **删除密码**:
    *   在弹出窗口中，找到您想删除的条目。
    *   点击该条目旁边的 **删除** 按钮。该条凭据会立即被移除。
